# Contributing

Thank you for considering a contribution.

These repositories are research projects. Contributions are most useful when they improve clarity, reproducibility, data quality, documentation, or tests.

## Useful contributions

- corrected references or DOI links
- reproducibility fixes
- documentation improvements
- dataset curation notes
- small examples that run cleanly
- issue reports with enough detail to reproduce the problem

## Pull requests

Please include:

1. what changed
2. why it changed
3. how it was tested
4. any limitations or assumptions

## Scientific claims

Do not add broad claims without a source. Link to a paper, DOI, dataset, or repository release when possible.
