# Contributing

Thank you for your interest in JAI Lab projects.

## Contribution areas

- documentation improvements
- examples and tutorials
- dataset formatting and validation
- benchmark definitions
- bug fixes
- reproducible notebooks
- scientific review and issue discussion

## Standards

Contributions should be reproducible, well-documented, and scientifically transparent. Please include enough context for others to understand the data, assumptions, methods, and limitations.

## Pull request checklist

- [ ] Clear description of the change
- [ ] Linked issue or motivation
- [ ] Reproducible example when relevant
- [ ] Documentation updated
- [ ] No private, sensitive, or restricted data included
