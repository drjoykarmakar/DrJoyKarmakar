# JAI Lab GitHub Kit

This package contains a complete GitHub organization/profile starter kit for **JAI Lab**.

## What is included

```text
.github/profile/README.md       Organization profile README
.github/ISSUE_TEMPLATE/         GitHub issue templates
.github/PULL_REQUEST_TEMPLATE.md Pull request template
CODE_OF_CONDUCT.md              Community standards
CONTRIBUTING.md                 Contributor guide
SECURITY.md                     Security policy
FUNDING.yml                     Funding placeholder
LICENSE                         MIT license placeholder
assets/svg/                     Hero, ecosystem, footer SVGs
branding/                       Brand guide and design tokens
docs/                           Roadmap and repository strategy
templates/                      README, dataset card, benchmark card templates
```

## How to upload

1. Create a GitHub repository named `.github` under your organization or user account.
2. Upload the contents of the `.github/` folder to that repository.
3. Upload `assets/`, `branding/`, `docs/`, and `templates/` if you want the full ecosystem kit preserved.
4. Edit links, repository names, and roadmap items as needed.

For an organization profile, GitHub displays `.github/profile/README.md` automatically.
