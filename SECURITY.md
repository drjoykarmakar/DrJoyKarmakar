# Security policy

JAI Lab repositories are research software projects. If you discover a security issue, please report it responsibly.

## Reporting

Open a private communication channel through the project maintainer or contact page at:

https://www.joykarmakar.com

Do not publicly disclose exploitable issues until they have been reviewed.

## Scope

Security reports may include:

- exposed credentials
- unsafe file handling
- dependency vulnerabilities
- notebook execution risks
- data leakage
- supply-chain risks

## Research data safety

Do not upload confidential, patient-level, proprietary, controlled, or otherwise restricted data unless explicit permission and legal basis exist.
