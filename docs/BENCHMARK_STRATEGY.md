# Benchmark strategy

## Why benchmarks

Benchmarks convert scientific ambition into measurable progress.

## Good benchmark tasks

A good benchmark has:

- a clear input
- a clear output
- a meaningful metric
- a realistic train/test split
- baseline models
- documented limitations
- relevance to real experiments

## Candidate benchmark types

### Molecular sensing

Predict fluorescence response, selectivity, limit of detection, or response class from molecular structure and experimental metadata.

### Fluorescent probes

Predict emission wavelength, quantum yield category, environment sensitivity, or analyte response.

### Spectroscopy

Classify or quantify analytes from spectral features with robust preprocessing.

### Medicinal chemistry

Predict activity class or potency for transporter inhibitor datasets.

## Baseline models

- random forest
- gradient boosting
- support vector machines
- graph neural networks when justified
- simple descriptor baselines
- uncertainty-aware variants

## Reporting

Each benchmark should include a benchmark card.
