# Community model

JAI Lab can grow from a personal research portfolio into a collaborative open research initiative.

## Contributor groups

- chemists contributing datasets or domain review
- ML researchers contributing models or benchmarks
- software engineers improving tooling
- students adding documentation and examples
- analytical chemists validating sensing workflows

## Contribution lanes

1. Documentation fixes
2. Dataset curation
3. Benchmark proposals
4. Example notebooks
5. Model baselines
6. Scientific review
7. Reproducibility checks

## Maintainer expectations

- Keep scientific claims grounded.
- Review data provenance carefully.
- Reject unsafe or unsupported chemical misuse content.
- Prefer transparent limitations over inflated performance.
