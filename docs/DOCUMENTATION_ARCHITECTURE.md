## Page template

Each program page should include:

1. scientific question
2. why it matters
3. what exists now
4. repositories
5. datasets
6. notebooks
7. roadmap
8. how to contribute

## Publication card template

Each paper should eventually get a page with:

- title
- journal/year
- role
- summary
- key scientific contribution
- related repository
- dataset/code link
- citation
