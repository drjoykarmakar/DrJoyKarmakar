# Launch checklist

Before making the new profile public, check:

## Profile

- [ ] Root README displays correctly.
- [ ] SVGs load in GitHub.
- [ ] Badges link correctly.
- [ ] GitHub activity cards load.
- [ ] No generic package instructions appear at the top of the profile.

## Claims

- [ ] Publication count is current.
- [ ] Citation metrics are current or marked with date.
- [ ] Awards are accurate.
- [ ] Repository descriptions are accurate.
- [ ] Roadmap items are not presented as completed.

## Repositories

- [ ] Pinned repos are ordered.
- [ ] Repo descriptions are updated.
- [ ] Topics are added.
- [ ] README templates are copied into major repos.
- [ ] Licenses are correct.

## Website

- [ ] Website matches GitHub positioning.
- [ ] GitHub links back to website.
- [ ] Website links back to GitHub.
