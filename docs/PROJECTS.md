# JAI Research Projects

JAI Research Projects is a public name for a set of research projects maintained by Joy Karmakar.

It should be described as a project collection or public workspace.

## Recommended wording

> JAI Research Projects is the name I use for a set of public research projects that connect synthetic chemistry, molecular sensing, spectroscopy, cheminformatics, and machine learning.

## Project descriptions

| Project | Suggested description |
|---|---|
| SensorGenome | Data schemas, benchmark ideas, and examples for molecular sensing experiments. |
| DyeMind | Notes and code for AI-assisted fluorescent probe and fluorophore design. |
| BimaneDB | Curated bimane dye table with descriptors and preliminary QSAR analysis. |
| NarcoticSense-AI | Spectroscopy preprocessing, chemometrics, and machine-learning classification for analytical chemistry research. |
| AZAI | Xylazine-related molecular and spectral data analysis. |
| pendrin-qsar | RDKit-based QSAR workflow for pendrin inhibitor datasets. |
| pendrin-hybrid | Combined ligand-based and structure-based modeling workflow for pendrin inhibitors. |
| cftr-qsar | QSAR notebooks for CFTR potentiator datasets. |
| Paper-Organizer | Local-first tool for organizing scientific PDFs and notes. |
