# Publication integration plan

## Goal

Connect publications to code, datasets, documentation, and future research programs.

## Recommended publication page fields

- Title
- Authors
- Year
- Journal
- DOI
- Scientific question
- Key result
- Role/contribution
- Related repository
- Related dataset
- Citation
- Reproducibility status

## GitHub linking pattern

For each paper, create:

```text
papers/<short-name>/README.md
papers/<short-name>/citation.bib
papers/<short-name>/figures/
papers/<short-name>/related-code.md
```
