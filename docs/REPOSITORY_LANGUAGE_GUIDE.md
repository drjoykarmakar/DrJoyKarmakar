# Repository language guide

Use words that match the current scope of the work.

## Prefer

- project
- repository
- tool
- workflow
- notebook
- dataset note
- benchmark note
- ongoing work
- public workspace
- research code
- exploratory analysis

## Avoid unless formally true

- division
- company
- official standard
- production-ready
- field-leading
- universal
- autonomous discovery
- operating system

## Replace phrases

| Instead of | Use |
|---|---|
| AI-driven discovery system | machine-learning workflow for a defined task |
| formal organization language | public set of research projects |
| autonomous sensor discovery | examples for screening candidate sensors |
| global community | contributors or users |
| production-grade tool | research prototype or analysis tool |
| benchmark standard | benchmark note or proposed benchmark task |
