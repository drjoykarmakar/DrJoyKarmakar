# JAI Lab Roadmap

## 2026 priorities

### Infrastructure

- Publish shared README standards.
- Create dataset and benchmark card templates.
- Standardize citation, licensing, and contribution language.
- Build common GitHub issue and pull request workflows.

### SensorGenome

- Define sensing experiment schema v1.
- Publish benchmark task definitions.
- Create uncertainty-aware model card format.
- Add active-learning workflow examples.

### DyeMind

- Define fluorescent probe dataset schema.
- Create baseline predictive modeling workflow.
- Create generative design evaluation template.

### Molecular Discovery Suite

- Publish QSAR workflow template.
- Publish docking workflow template.
- Create ADMET and SAR report templates.

### Open Science Tools

- Publish BimaneDB structure.
- Publish paper organization workflow.
- Create reusable documentation system.

## Long-term vision

Make molecular discovery as reproducible, programmable, and collaborative as modern software development.
