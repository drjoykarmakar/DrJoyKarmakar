# Upload Instructions

For a personal GitHub profile README:

1. Create a public repository named exactly `drjoykarmakar`.
2. Upload `README.md` to the root of that repository.
3. Upload the `assets/` folder to the root of that repository.
4. Commit the files.
5. GitHub will render the README on `https://github.com/drjoykarmakar`.

Optional but recommended:

- Copy `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and `SECURITY.md` into major repositories.
- Use `templates/REPOSITORY_README_TEMPLATE.md` for each scientific software repository.
