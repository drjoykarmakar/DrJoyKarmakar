# Upload instructions

For a personal GitHub profile README:

1. Create or open a repository named exactly `drjoykarmakar`.
2. Put `README.md` at the root of that repository.
3. Upload the `assets/` folder at the root so the SVG paths work.
4. Commit and push.

For community files across repositories:

1. Keep `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and `SECURITY.md` in the root of major repositories.
2. Copy `.github/ISSUE_TEMPLATE/` and `.github/PULL_REQUEST_TEMPLATE.md` into active repositories.
3. Use the templates in `templates/` for new project READMEs, dataset cards, and benchmark cards.
