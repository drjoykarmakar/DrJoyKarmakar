# Verify before claiming

Before adding a claim to a public profile, check whether it can be verified.

## Good sources

- journal pages
- DOI pages
- Google Scholar profile
- university directory or department page
- award announcement page
- conference program
- repository release page

## Claims to avoid or qualify

- citation counts without a date
- rankings
- broad impact statements
- “first,” “largest,” or “best” claims
- accepted/submitted manuscripts without a public preprint or citation
- roles that are informal rather than official

## ORCID note

Add ORCID only after confirming the exact record. If the record cannot be clearly verified, leave it out of the README and use Google Scholar and DOI links instead.
