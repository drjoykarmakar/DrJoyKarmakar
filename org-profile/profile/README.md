## Mission

JAI Lab builds open datasets, benchmarks, scientific software, and AI systems for molecular discovery.

The goal is to make molecular discovery more reproducible, programmable, benchmarkable, and useful for real chemistry.

## Operating principle

> Better molecular AI requires better scientific infrastructure.

JAI Lab is built around structured experimental data, reproducible benchmarks, uncertainty-aware modeling, and open documentation.

---

## Repository standard

Every public repository should answer:

1. What scientific problem does this solve?
2. What can a user run today?
3. What data, model, or benchmark is included?
4. How can the work be reproduced?
5. How should users cite it?
6. What are the limitations?
