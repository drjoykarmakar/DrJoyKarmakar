## Why this exists

Emerging adulterants require analytical tools that are transparent, reproducible, and scientifically grounded. AZAI focuses on xylazine-centered molecular intelligence and sensing workflows.

## What this repository should contain

- xylazine analytics notes
- candidate molecular sensing workflows
- computational chemistry utilities
- spectroscopy and chemometrics links
- safety-conscious documentation

## Quickstart

```bash
git clone https://github.com/drjoykarmakar/AZAI.git
cd AZAI
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
